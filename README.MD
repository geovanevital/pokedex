How to run
==========

Start metro bundler

```shell script
react-native start
```

Run project in Android environment

```shell script
react-native run-android
```

Run project in iOS environment

```shell script
react-native run-ios
```

Alias to run
============

Start metro bundler

```shell script
yarn start
```

Run project in Android environment

```shell script
yarn android
```

Run project in iOS environment

```shell script
yarn ios
```
